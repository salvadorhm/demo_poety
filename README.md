# Proyecto demo de Python y Poetry

## 1. Inicializar proyecto


```bash
poetry init
```

## 2. Importar librerias


```bash
xargs poetry add < requirements.txt
```

## 3. Inicializar el shell

```bash
poetry shell
```

## 4. Ejecutar sin entrar al shell

```bash
poetry run python3 app.py
```

## 5. Salir del shell

```bash
exit
```
